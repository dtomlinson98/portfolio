Self Reflection

These tests identified numerous insights that can be used to improve the user experience. It is pivotal to establish a clear purpose for the site. Adding "Shop" or "Buy Now" buttons would go a long way in placing the user where they want to be to accomplish their task. Also, reducing the amount of clicks a user has to perform is very important. The amount of confusion and mental load the excess clicks cause is enough to drive many users away from the website. Next, the addition of relevant informaiton would be helpful in leading the consumer to a decsion. This informaiton should be made clear and availble without the user having to look for it. The site should anticipate the users questions and answer them with haste. The Snap On website is a great case for a large brand that completely misses the mark when it comes to user experience on their website. However, there are many great lessons that can be learned from this website.
