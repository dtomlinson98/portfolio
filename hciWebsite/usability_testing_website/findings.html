Purpose is Unclear

Too Many Clicks

The user has to click 6 times just to add a hammer to her cart. The page requires too many actions from the user to accomplish their task.

Confusing Information

When the user selects the "Find a Franchise" link, the information on the page caused a moment of hesitation and forced a choice on the user. After making a choice the information th user needed was not made available.

Lack of Relevant Information

Information like product reviews are not readily available on the site. The user expected to find a review under the product on the product page. This information was very difficult to find for the average user.

User 2

Purpose Is Not Clear

The main purpose of the website, which is to buy tools, is not abundantly clear at first glance. A shop or buy now button should be added to the main screen to guide users where the company wants them to go.
Watch Here

Irrelevant Screen

The screen that asks uses to specify the type of consumer they are is irrelevant and confusing to many users. The same information is shown regardless of the users choice. This screen only causes potential confusion and should be omitted.
Watch Here

Search Bar

The search bar does not display relevant results based on the users query. The only information that is displayed is for products. Search results should include products as well as pages and tools the user can use.
Watch Here

Guest Option

There is no guest option requiring the user to create an account. Multiple steps are involved with this step so this presents a time where the user may become disinterested and not purchase the product. A guest option should be added to decrease the barrier between the consumer and a purchase.
