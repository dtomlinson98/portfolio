Heursitc Evaluation

Usability Problem Heuristic(s) Violated Severity Level
The purpose of the website is not immediately clear. There is not one action that is used as the forefront of the page. Use Familiar, Consistency, Recognition Over Recall 3
Website has a lot of wasted space by including irrelevant images and text. Clean and Functional 2
Navigation is in a common location on the page but has misleading and confusing page names. Freedom, Recognition Over Recall, Clean and Functional 3
The product lists are only accessible by following a fixed path down a category tree. Freedom 2
Once arriving at a product list, there is no way to filter the results. Clean and Functional, Freedom, Flexibility 3
The image carousel on the main page does not work automatically and must be scrolled manually. Clean and Functional 3
There is no footer at the bottom of the main page causing me to think there was some kind of issue not allowing me to scroll all the way to the bottom of the page. Consistency, Use Familiar 1
